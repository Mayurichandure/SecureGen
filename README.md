# SecureGen

A secure code generation and analysis tool built with modern web technologies.

## Features

- Code analysis and security scanning
- Secure code generation
- User-friendly interface with React and TypeScript

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd SecureGen
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Technologies Used

- **Frontend**: React 18, TypeScript, Vite
- **UI Library**: shadcn/ui, Radix UI components
- **Styling**: Tailwind CSS
- **Backend**: Supabase (for database and serverless functions)
- **Testing**: Vitest, Testing Library
- **Linting**: ESLint

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run test` - Run tests
- `npm run lint` - Run ESLint

## Deployment

This project can be deployed to platforms like Vercel, Netlify, or any static hosting service.

For Vercel:
1. Connect your GitHub repository to Vercel
2. Deploy automatically on push
